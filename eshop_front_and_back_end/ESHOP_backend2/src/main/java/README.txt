1. Börja med endpointen "/addproducts" för att lägga till producter
2. Sen man  söker alla producter om man vill, med end pointen "/getproducts"
3. Sen kan man lägga till order med "/addorder/{email}/{product}/{quantity}"
	email=beställarens e-post, product=productID, quantity=anatalet(siffra) som man vill beställa.

4. Sen kan man söka på alla ordra med "/getorders"
